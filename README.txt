UDS Field Log Mobile Web App

No Node.js required.
No npm required.
No programs required on your work computer.

How to use on iPhone:
1. Upload these files to a simple web host such as Netlify, GitHub Pages, Vercel, or company SharePoint/web server.
2. Open the website link in Safari on your iPhone.
3. Tap Share.
4. Tap Add to Home Screen.
5. Open UDS Log from your Home Screen.
6. Load it once while online. After that, it can work offline.

Files included:
- index.html
- style.css
- app.js
- manifest.json
- service-worker.js

Current features:
- Offline field entries
- Camera/photo attachment
- DS/NS shift
- Area / heading
- Work type
- Job being performed
- Next shift
- PTHA / LIF / housekeeping / barricades / ground checks
- Action register
- Daily report
- Save as PDF using iPhone print/share
- CSV action export
- Backup JSON export

Recommended easiest hosting:
Netlify drag-and-drop:
1. Go to app.netlify.com/drop
2. Drag the whole unzipped folder onto the page
3. Netlify gives you a website link
4. Open the link on iPhone Safari
5. Add to Home Screen
